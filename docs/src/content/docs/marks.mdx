## Usage

Each mark ships as a standalone SVG in `src/marks/` with its tool hue baked in — safe to use directly as a favicon, in an OG image template, or in a header lockup:

```html
<span class="c-logo">
  <span class="co">erisera</span><span class="tool">matey</span><span class="ver">v4.2.0</span>
</span>
```

The favicon and OG-image treatments are demonstrated on the [showcase](/showcase/) page.
