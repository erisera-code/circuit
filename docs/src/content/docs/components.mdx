## Admonitions — Starlight's own

Starlight ships `note` / `tip` / `caution` / `danger` asides natively. Circuit doesn't reimplement them — it remaps their colors (see [Color](/color/)) so they use the same fixed semantic palette as everything else in the family.

<Aside type="note">Uses the live accent — the only admonition type that changes per tool.</Aside>
<Aside type="tip">Always success-green, everywhere.</Aside>
<Aside type="caution">Always amber, everywhere.</Aside>
<Aside type="danger">Always error-red, everywhere.</Aside>

```mdx
import { Aside } from '@astrojs/starlight/components';

<Aside type="tip">Always success-green, everywhere.</Aside>
```
